## ğŸš€ Setup Instructions

### 1. Database Setup
1. Install MySQL and MySQL Workbench.
2. Import the database dump from the `db/` folder into your MySQL instance.
3. Update database credentials in the backend configuration if necessary.

### 2. Backend Setup
1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   # Or individually:
   # pip install mysql-connector "fastapi[all]"
   ```
3. Start the server:
   ```bash
   uvicorn main:app --reload
   ```

### 3. Dialogflow Setup
1. Create a new Agent in Google Dialogflow.
2. Import the intents and entities from `dialogflow_assets/`.
3. Configure the **Fulfillment** webhook URL to point to your backend (see Ngrok setup).

### 4. Ngrok Setup (for HTTPS Tunneling)
Since Dialogflow requires a public HTTPS URL for webhooks, use Ngrok to expose your local server.
1. Download [Ngrok](https://ngrok.com/download).
2. Run Ngrok:
   ```bash
   ngrok http 8000
   ```
3. Copy the HTTPS URL generated by Ngrok and paste it into the Dialogflow Fulfillment Webhook URL field.

### 5. Frontend Setup
1. Open the `frontend/` folder.
2. Launch `index.html` in your browser to test the chatbot integration.
